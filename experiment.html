What I'm trying out

Vanilla-JS port of the React/GSAP CardSwap. Same elastic timeline. Tunable in seconds.

Knobs we can play with

  • delay — how long each card stays in front (currently 4000ms)
  • cardDistance / verticalDistance — how spread out the stack feels (currently 40 / 40)
  • skewAmount — the tilt that creates depth (currently 5°)
  • easing — elastic (soft, bouncy, ~2s per move) or linear (snappy, ~0.8s)
  • card content — month chip, logo, name, prize line, tags. Easy to redesign.
  • card backgrounds — soft pastel gradients (sage / lavender / coral / blush / paper)

Where this could land on the main site

  • Replace the "marquee headlines" horizontal scroll — same purpose (featuring top comps), more dynamic.
  • Replace the three-phone preview — the swap IS the "product preview" since it shows real comps.
  • Add as a new section above the catalog — "this week's spotlight" feel, then catalog below.
  • Keep both — swap as hero centerpiece, horizontal scroll as quick-scan below.
